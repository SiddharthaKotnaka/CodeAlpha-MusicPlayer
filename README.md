# 🎵 NeonCat Music Player

A modern and responsive music player built using HTML, CSS, and JavaScript with cinematic visuals and interactive audio controls.

## 🚀 Features

- Play/Pause Music
- Next & Previous Controls
- Interactive Playlist
- Dynamic Progress Bar
- Volume Control
- Animated Equalizer
- Glassmorphism UI
- Live Background Visuals
- Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Remix Icons
